IMAGE QUALITY TIPS:
-------------------
Compact (80x80):
  - Start with 400px+ source image
  - Use "Bicubic Sharper" resampling when resizing
  - Apply slight sharpening after resize
  - Save as PNG (not JPG)

Full (400x560):
  - Start with 1200px+ source image
  - Use "Bicubic" resampling
  - Save as PNG for crisp text

NAMING: Use exact card ID + .png extension
